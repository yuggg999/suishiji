## 核心能力

- **自动记账** — 通过无障碍服务自动识别微信/支付宝支付成功页面，提取金额、商户、自动分类
- **手动记账** — 支持手动添加收支记录，带分类、日期、备注、精确定位
- **支付通知监听** — 通过 NotificationListenerService 捕获刷脸/指纹支付后的通知，兜底记账
- **退款识别** — 自动识别支付宝/微信退款通知，自动记录
- **位置记录** — 集成高德地图 SDK，每笔交易记录地理位置，地图展示
- **统计图表** — 月度日历、分类饼图/条形图、每日支出排行
- **固定支出** — 管理每月固定支出（房租、订阅等），自动计入月结
- **数据导入导出** — CSV 导入导出
- **深色模式** — 支持明暗主题切换
- **月度预算** — 设置预算，实时跟踪进度
